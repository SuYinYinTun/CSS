RAM , set at the memory slot on motherboard , has no data when power-off. By the time the power supports, the electric charge flows to the RAM and then start to be carrying instruction and storing data. As soon as the computer kept in shut-down state, the all data on RAM will disappear since it is called the short term memory of computer. The label on memory module describes the type and speed of memory chip and the name of manufactured company.
